# smart-campus-information-system
SMART CAMPUS INFORMATION SYSTEM

This project is a Python-based campus management system that includes:
- Student Registration
- Course Enrollment
- Sorting & Searching
- Fee Calculation
- File Handling
- Directory Scanning
- Performance Analytics using Pandas & Matplotlib

How to Run:
1. Install Python
2. Install required libraries:
   pip install numpy pandas matplotlib
3. Run:
   python smart_campus.py
